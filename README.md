# Torre de Controle Logístico: Gestão de Performance de Transporte

Resumo: Análise de eficiência operacional focada no monitoramento de indicadores de nível de serviço (OTIF, On Time e In Full). O dashboard integra volumetria expedida, peso bruto e faturamento, permitindo o acompanhamento geográfico por região e a avaliação de performance individualizada por transportadora e cliente.

Tecnologia: Microsoft Excel
